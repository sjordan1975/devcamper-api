# DevCamper API

> Backend API for th DevCamper Application

## Usage

Rename "config/config.env.template" to "config/config.env" and update values/settings to your own

## Install dependencies

```
npm install
```

## Build App

```
npm run build
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm run start
```
